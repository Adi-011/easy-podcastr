Prerequisites:
- Git
- Node.js
- npm 

Install the project dependencies using npm:
- npm install

Set Up Environment Variables:

  CONVEX_DEPLOYMENT=

  NEXT_PUBLIC_CONVEX_URL=

  NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=

  CLERK_SECRET_KEY=

  NEXT_PUBLIC_CLERK_SIGN_IN_URL='/sign-in'

  NEXT_PUBLIC_CLERK_SIGN_UP_URL='/sign-up'






Running the Project:

npm run dev
